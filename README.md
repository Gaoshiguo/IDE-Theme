# IDE-Theme
pycharm webstorm IDE color Theme
#可适用于*Pycharm*、*Webstorm*、*IntelliJ IDE*
***
## 使用方法：
1、先下载相应的jar包
2、在相应的IDE开发工具中，File->Settings->color theme->import themem ,然后根据jar包位置选择相应的路径，最后apply，重启完成！
